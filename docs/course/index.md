## How to use the course

Read modules in numerical order. Inside a module, use the left sidebar to move between chapters. Use the previous and next links at the bottom of a chapter for continuous reading. Return to this dashboard only when changing modules.

The formal syllabus begins with Module 01. Module 00 is a preparatory chapter that establishes the vocabulary and study habits used throughout the book.
